基于我为你构建的 **XAUUSD 高频量子交易系统方案**，我们在系统中并非一次性引入全部 101 个因子，而是采用了 **“特征降维” (Feature Reduction)** 策略。

具体来说，系统引入了 **5 到 8 个核心输入因子** 进入量子电路。

### 为什么不是 101 个？

直接将 Alpha101 的 101 个因子全部输入当前的量子神经网络（QNN）是不可行的，主要原因有二：

1. **贫瘠高原（Barren Plateaus）**：量子电路的参数越多，梯度消失问题越严重，导致模型无法训练 。


2. **噪声累积**：在 NISQ（含噪声中尺度量子）时代，电路深度有限，过多的输入特征会导致计算信噪比急剧下降。

### 具体引入了哪几个关键因子？

根据研究报告中的 **XAUUSD 核心特征集 (Section 4.1)** 和代码配置，我们实际上精选了以下 **5 个物理意义明确的核心因子**（映射到 5-8 个量子比特）：

1. **动量因子 (RSI / Momentum)**
* **物理含义**：通过 `RSI` 或动量指标映射。
* **量子映射**：映射为量子旋转门的 **旋转角度 **。


2. **趋势强度 (EMA Spread)**
* **物理含义**：快速均线与慢速均线的差值，代表趋势的强弱。
* **量子映射**：映射为量子态的 **相位 **（正负对应多空趋势）。


3. **市场拒绝/不确定性 (Wick Ratio)**
* **物理含义**：K 线影线与实体的比例。在黄金中，长影线代表强烈的市场拒绝（Rejection）。
* **量子映射**：映射为 **叠加态振幅**（高影线比率 = 高不确定性 = 叠加态）。


4. **相变信号 (Volume Shock)**
* **物理含义**：成交量突变。黄金行情的启动往往伴随着成交量的非线性爆发。
* **量子映射**：作为 **控制参数**，用于触发量子电路中的纠缠门（CNOT）。


5. **微观结构压力 (DOM Pressure Proxy)**
* **物理含义**：买卖盘口的不平衡压力（由于 MT5 无真实 L2 数据，使用算法近似）。
* **量子映射**：映射为 **纠缠强度**。
