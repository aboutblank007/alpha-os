## 🔧 关键参数调整

### 1. 保守的保证金设置
```python
margin_per_lot = 5000.0  # 从 $1000 增加到 $5000
```

**效果**: 同样的Kelly%，手数变为原来的 1/5

**示例**:
- Kelly% = 5%, 账户 $10,000
- 之前: risk=$500, lot=500/1000=0.50手
- 现在: risk=$500, lot=500/5000=0.10手 ✅

### 2. 最小手数提升
```python
min_lot = 0.05  # 从 0.01 提升到 0.05
```

**原因**: 高频交易中，过小的手数（0.01-0.04）意义不大

### 3. 0.05手步进
```python
lot_size = round(lot_size / 0.05) * 0.05
```

**效果**: 只产生 0.05, 0.10, 0.15, 0.20, ..., 0.95, 1.00

## ⚙️ 可调参数

### 用户可在 `automation_rules` 表调整:

**kelly_fraction** (凯利分数):
- 0.10 = 极保守 → 手数更小（0.05-0.40范围）
- 0.25 = 保守（默认）→ 0.05-1.00范围
- 0.50 = 激进 → 更大手数，容易触碰1.0封顶

**max_lot_size** (最大手数):
- 0.50 = 半手封顶（极保守）
- 1.00 = 标准（默认）
- 2.00 = 允许更大仓位（不推荐高频）

**kelly_lookback_trades** (回看交易数):
- 30 = 快速适应（波动大）
- 50 = 平衡（默认）
- 100 = 稳定（变化慢）
